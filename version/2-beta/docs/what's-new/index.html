# What's New

ProTips™ Why v2?

NotifyBC has been built on Node.js LoopBack (opens new window) framework since 2017. LoopBack v4, which was released in 2019, is backward incompatible. To keep software stack up-to-date, unless rewriting from scratch, it is necessary to port NotifyBC to LoopBack v4. Great care has been taken to minimize migration effort.

# v2.0.0

See Migration Guide for more information.

  • Runs on LoopBack v4
  • All code is converted to TypeScript
  • Upgraded OAS (opens new window) from v2 to v3
  • Docs is converted from Jekyll to VuePress
