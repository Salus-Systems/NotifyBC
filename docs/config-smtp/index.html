# SMTP

By default NotifyBC acts as the SMTP server itself and connects directly to recipient's SMTP server. To setup SMTP relay to a host, say smtp.foo.com, add following smtp config object to /src/config.local.js

module.exports = {
  ...
  smtp: {
    host: 'smtp.foo.com',
    port: 25,
    pool: true,
    tls: {
      rejectUnauthorized: false
    }
  },
}

Check out Nodemailer (opens new window) for other config options that you can define in smtp object. Using SMTP relay and fine-tuning some options are critical for performance. See benchmark advices. There are also options allowing you to throttle down throughput if needed.
