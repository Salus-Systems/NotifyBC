Versatile

  • Anonymous or CA SiteMinder authenticated subscriptions
  • Push and in-app pull notifications
  • Email and SMS push notification channels
  • Unicast and broadcast message types
  • User specifiable filter rules
  • Notification auto-gen from RSS

Non-intrusive

  • Handles common backend business logic only, allowing site developer implement user subscriptions or display in-app notifications using UI widgets native to the site.
  • Loose coupling - interacts with user browser or other server components through RESTful API.

Microservice

Containerized deployment to PAAS such as OpenShift. No need for complex enterprise-wide shared service. Better meet privacy mandates.
